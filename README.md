# 2D-JGame

Trying to make a 2D platformer game built with Unity.

## Game Description
A two-dimensional platformer game featuring a player character that can move left and right, jump, and perform various animations. The game is built with the Unity engine and uses Unity's 2D physics system.

## Features
- Smooth movement with A/D keys
- Jump with Spacebar
- Animation system (Idle ✓, Walk ✕, Jump ✕, Falling ✕ etc.)
- Ground detection for jumping
- Realistic physics with Rigidbody2D

## Technologies
- Unity Engine
- C#
- Unity 2D Physics
- Unity Animator System

## Installation
1. Clone the project: `git clone https://github.com/Jaromix3/2D-JGame.git`
2. Open the project in Unity Hub
3. Run the main scene

## Controls
- A - Move left
- D - Move right
- Space - Jump

## Development Status
The project is in early development stages. Planned future features include enemies, collectibles, and additional levels.
